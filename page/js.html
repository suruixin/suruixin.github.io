localCompare()

用本地特定的顺序来比较两个字符串

function local(a,b){ return a.localeCompare(b) } local("a","c"); //-1 local("a","a"); //0 local("c","a"); //1
toLowerCase()

大写转小写

var str = "Hello World"; str.toLowerCase();//'hello world'
toUpperCase()

小写转大写

var str = "Hello World"; str.toUpperCase();//"HELLO WORLD!"
in

x in y 如果x可以转换为字符串并且y为一个对象，那么判断y中是否有x如果有则返回true，否则返回false.

"x" in {x:1};//true; "y" in {x:1};//false
instanceof

a instanceof b 左侧接收一个对象，右侧若为左侧的类，若满足则返回ture，否则返回false

var time = new Date(); time instanceof Object;//true time instanceof Array;//false
